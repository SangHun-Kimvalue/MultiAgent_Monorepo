## Gate Rule

`DISCOVERY_PASS` requires:
- zero `TBD` items
- zero open `Evidence Required` items tied to the core value proposition
- zero unverified external adapter/tool contracts that carry core functionality
- no unresolved P1 Senior Critique finding

`Assumption` items may remain only when non-blocking and impact-if-wrong is documented.
`Decision` items may remain only when rationale and alternatives are recorded. If the decision downgrades the value proposition, user acceptance is required.
